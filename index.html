Python is basically executable psuedocode.

The print statement is used to send data to the console.

Print can be used to send strings, numbers, or any other data type to the console.

Variables
foo = "text"
print foo
>>> text
bar = 1
print bar
>>> 1
lorem = []
ipsum = None
print lorem, ipsum
>>> []
>>> None

A variable is an identifier that can be assigned any value in python.

Variables can be used in equations, boolean operators, or in any manner a regular data type can be used.

foo = 1
bar = 2
foobar = foo + bar
print foobar
>>> 3
Comments

Comments are useful information about your code to help clarify it and help others read it.

Comments are ignored by the python interpreter.

In python, the '#' character is used to denote a comment.

#This data isn't read by the interpreter
Operators

Operators are traditional mathematical operators, with a few programming specific ones added in.

  • + adds numbers, or concatenates strings
  • - subtracts numbers
  • * multiplies numbers
  • / divides numbers, will only give decimal precision if one of the numbers divided was a decimal to begin with
  • % modulo, like '/' but returns the remainder
  • ** exponents, so 2 ** 2 is the same as 22
Strings

Strings are immutable arrays of bytes representing unicode characters.

Strings can be indexed, just like a list.

foobar = "foobar"
print foobar[0:3]
>>> foo
print foobar[3:]
>>> bar
Lists

Lists are indexable,mutable data structures that can store any data type.
